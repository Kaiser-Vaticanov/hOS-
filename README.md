\# hOS

A minimalistic OS based on FASM.

\## Commands

\- `exit` — shuts down the system.

\- Any text — input with Backspace support.

\## How to build

1\. Install FASM.

2\. Run: `fasm hos.asm hos.bin`

3\. Run in QEMU: `qemu-system-i386 -drive format=raw,file=hos.bin`
